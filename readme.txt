პროექტში განხორციელებულია ბაზასთან დაკავშირება, ინფორმაციის წაკითხვა სხვადასხვა ფილტრაციის კრიტერიუმით, მაგალითად, მათემატიკაში მაღალი ქულების მქონე სტუდენტები, ასევე ჩანაწერების დამატება, განახლება და წაშლა input ფუნქციის საშუალებით. მონაცემთა ვიზუალიზაცია განხორციელდა `matplotlib` ბიბლიოთეკის გამოყენებით. აგებულია სამი განსხვავებული დიაგრამა: მათემატიკის ქულების ჰისტოგრამა, ბარის დიაგრამა წერის საშუალო ქულების მიხედვით სქესის მიხედვით, და წრიული დიაგრამა ტესტის მომზადების კურსის კატეგორიების მიხედვით.
